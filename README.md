# Template solutions

This is a collection of template solutions that can be used as starting points for building a web project.  There are a number of different solutions with slight differences which may be mroe of less suitable depending on your planned project.

Below is a list of different solutions and their distinctive characteristics.

## 1. CodeFirstMVCWithUnitOfWorkAndRepositories

- ASP.Net MVC 5
- Bootstrap 3 (themed with Bootswatch Cerulean)
- Entity Framework 6, Code First with Manual Migrations (SQL Server)
- Log4Net
- Web project unit test suite
- AutoFac
